# Library-PHP
Ruslan Rohovskyi – C24741821 – TU856/2
Library PHP CA project 
1. Pages structure:
Home
Search
Library
My Reservations (if user logged in)
Login (if no user logged in) 
Register (on Login page)
Account (if user logged in)
Logout (on Account page)

2. Features
Responsive layout (mobile and tablet versions), implemented Burger menu
Modularity for templates and includes
Simple and user-friendly interface

3. Project structure
Assets for styles, images and scripts
Config for Database connection, shortcuts for PATH includes
Includes for repetitive includes like header or meta (php headers for session)
Queries for Database queries and search requests
Templates for Results output or Search filters after running queries
